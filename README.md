# TD Classic

Egy könnyed, böngészőben futó tower defense minijáték. Indítsd a hullámokat, építs Lángtornyokat, és védd meg a kristályt a nyolc eltérő ellenfél-hullámtól.

## Futtatás
Nyisd meg az `index.html` fájlt egy modern böngészőben (asztali környezet ajánlott), és kattints a pályára tornyok lerakásához.
